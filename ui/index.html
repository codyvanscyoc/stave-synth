OUTPUT TRIM
Pre-Limiter Trim 2.0x
Drive into the soft limiter. Lower = cleaner/quieter. Higher = louder, more saturated. Can't clip (tanh).
FADERS
OSC 1 — Pad volume. ALT: OSC 2 volume.
PIANO/ORGAN — Volume. ALT cycles: Tone (high cut) → Compressor/Leslie.
FILTER — Lowpass cutoff (standalone, no ALT).
FX — Reverb dry/wet. ALT: Shimmer volume.
MASTER — Master output volume.
PRESETS
Tap empty slot — Save current settings.
Tap filled slot — Load that preset.
Long-press filled slot — Overwrite with current settings.
Double-tap filled slot — Shows X. Tap X to delete, tap elsewhere to cancel.
MIDI LEARN
Tap MIDI indicator — Enter learn mode (shows "MAP").
Tap a fader — Select it for mapping.
Turn a knob/slider — Maps that CC to the selected fader.
Tap MIDI again — Exit learn mode.
TOP BAR
OSC1 / OSC2 — Tap to mute/unmute.
PIANO/ORGAN — Tap to cycle: Piano → Organ → Off.
SHIM — Toggle shimmer (octave-up shine on pad).
FRZ — Freeze reverb tail (appears when shimmer is on).
T: +/- — Transpose all MIDI input in semitones.
STOP — Panic: kills all notes and audio.
OCTAVE
OSC 1 fader +/- — Shift OSC1 octave (ALT mode shifts OSC2).
PIANO fader +/- — Shift piano octave independently.
